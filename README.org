* Description
A simple CSV parser based on janet's peg library.

It can parse:

+ headers
+ multiline fields
+ escaped field characters

* API

+ ~(csv-parser/parse data &opt header)~
  + ~data~ :: a string
  + ~header~ :: use the first line as an header

Returns an Array of Arrays or an Array of Dicts.
